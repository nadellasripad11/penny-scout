# SerenityOS - Web-Based Operating System

## Project Description

SerenityOS is a fully functional web-based operating system built with vanilla JavaScript, HTML, and CSS. It features a modern dark-themed interface with a glass-morphism design, complete with draggable windows, a real-time clock, and 8 interactive applications.

### Key Features

**Core OS Functionality:**
- Personalized welcome screen with name input
- Dynamic desktop with persistent topbar and taskbar
- Real-time clock that updates every second
- Window management system with draggable windows and z-index layering
- Window close functionality with clean DOM removal

**Built-in Applications (8 Total):**
1. **About Me** - Personal profile with skills and bio
2. **Projects** - Showcase of project cards with descriptions and tags
3. **Portfolio** - Professional work timeline and project overview
4. **Contact** - Email form for messaging with contact information
5. **Notes** - Note-taking app with localStorage persistence
6. **To-Do** - Task management with add/delete functionality
7. **Calculator** - Full-featured calculator with operations (+, -, ×, ÷)
8. **Timer** - Pomodoro timer (25:00) with Start/Pause/Reset controls

### Technical Highlights

- **Frontend Architecture**: Vanilla JavaScript DOM manipulation
- **Styling**: CSS with glass-morphism effects, animations, and responsive dark theme
- **State Management**: localStorage for persistent data (notes, to-do items)
- **Event Handling**: Dynamic event listeners for all interactive elements
- **Performance**: Lightweight, no frameworks or dependencies
- **Deployment**: Vercel static site hosting with automatic GitHub integration
