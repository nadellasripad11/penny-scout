## Tips for Best Results

- Use the longer prompt for more detailed results
- Avoid vague terms, be specific about layout and colors
- If first result isn't perfect, regenerate 2-3 times
- Look for images that show:
  - Clean, organized window layout
  - Visible text in UI elements
  - Professional color scheme
  - Modern design aesthetic
  - Good contrast and visibility

## What Makes a Good Banner

✅ **Good:**
- Shows the OS interface clearly
- Multiple windows visible
- Clear topbar and taskbar
- Professional appearance
- Dark theme with purple accents
- Modern, polished look

❌ **Avoid:**
- Too much text
- Blurry or unclear details
- Cartoonish style
- Inconsistent UI elements
- Poor color choices
